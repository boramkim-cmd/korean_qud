# 번역 품질 보증 체크리스트

이 문서는 Korean Localization 모드의 품질을 검증하기 위한 체크리스트입니다.

## 🎮 기능 테스트

### 모드 로딩
- [ ] 모드가 Mods 메뉴에 표시됨
- [ ] 모드 활성화 시 오류 없음
- [ ] 게임 재시작 후 모드가 정상 작동
- [ ] 로그 파일에 오류 메시지 없음

### UI 번역
- [ ] **메인 메뉴**
  - [ ] "New Game" → "새 게임"
  - [ ] "Continue" → "계속하기"
  - [ ] "Options" → "옵션"
  - [ ] "Mods" → "모드"
  - [ ] "Quit" → "종료"

- [ ] **캐릭터 생성**
  - [ ] Genotype 설명 한글 표시
  - [ ] Calling 설명 한글 표시
  - [ ] Attribute 이름 및 설명 한글 표시
  - [ ] Skill 이름 및 설명 한글 표시

- [ ] **인벤토리**
  - [ ] 아이템 카테고리 한글 표시
  - [ ] 아이템 설명 한글 표시
  - [ ] 무게/가치 레이블 한글 표시

- [ ] **옵션 메뉴**
  - [ ] 모든 설정 항목 한글 표시
  - [ ] 설정 설명 한글 표시
  - [ ] 드롭다운 메뉴 한글 표시

- [ ] **대화 시스템**
  - [ ] NPC 대화 한글 표시
  - [ ] 선택지 한글 표시
  - [ ] 퀘스트 텍스트 한글 표시

### 한글 조사 처리

## 🔧 기술 검증

### XML 구문
- [ ] 모든 XML 파일 구문 오류 없음
- [ ] 태그 올바르게 닫힘
- [ ] 속성 값 올바르게 인용됨
- [ ] 특수 문자 올바르게 이스케이프됨

### 게임 안정성
- [ ] 게임 크래시 없음
- [ ] 메모리 누수 없음
- [ ] 성능 저하 없음
- [ ] 세이브/로드 정상 작동

### 호환성
- [ ] 최신 게임 버전과 호환
- [ ] 다른 모드와 충돌 없음
- [ ] 모든 플랫폼에서 작동 (Windows/macOS/Linux)
