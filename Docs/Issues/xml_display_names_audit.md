## 주의사항

### 컬러 태그 포함 항목
많은 항목이 `{{color|text}}` 형식의 컬러 태그를 포함합니다:
- `{{crochide|croccasins}}`
- `{{K|fullerite}} boots`
- `{{zetachrome|zetachrome}} pumps`

번역 시 셰이더명은 유지하고 내용만 번역해야 합니다:
```
{{crochide|croccasins}} → {{crochide|악어신발}}
```

### 복합어 항목
다중 단어로 구성된 항목들:
- "energy relay" (에너지 중계기)
- "failed energy relay" (고장난 에너지 중계기)
- "dried lah petals" (말린 라 꽃잎)
