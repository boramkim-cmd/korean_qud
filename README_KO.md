# Caves of Qud 한글화 모드

Caves of Qud의 완전한 한글 번역 모드입니다.

## 주요 기능

- ✅ **완전한 UI 번역** - 메인 메뉴, 인벤토리, 캐릭터 화면, 옵션 등
- ✅ **한글 조사 자동 처리** - 자연스러운 한글 문법을 위한 조사 자동 변환
- ✅ **대화 및 퀘스트 번역** - 대화, 퀘스트, 스토리 콘텐츠
- ✅ **캐릭터 생성** - 유전형, 소명, 능력치, 스킬 번역

## 설치 방법

### Steam Workshop (권장)

1. [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=XXXXXXXXX)에서 모드 구독
2. Caves of Qud 실행
3. Mods 메뉴에서 모드 활성화
4. 게임 재시작

### 수동 설치

1. [GitHub Releases](https://github.com/boramkim-cmd/korean_qud/releases)에서 최신 버전 다운로드
2. Caves of Qud Mods 폴더에 압축 해제:
   - **Windows**: `%USERPROFILE%\AppData\LocalLow\Freehold Games\CavesOfQud\Mods\`
   - **macOS**: `~/Library/Application Support/com.FreeholdGames.CavesOfQud/Mods/`
   - **Linux**: `~/.config/unity3d/Freehold Games/CavesOfQud/Mods/`
3. Caves of Qud 실행
4. Mods 메뉴에서 "Korean Localization (한글화)" 활성화
5. 게임 재시작

## 알려진 문제

- 일부 절차적 생성 콘텐츠는 여전히 영어로 표시될 수 있습니다
- 역사 텍스트 생성은 영어 이름을 사용합니다 (의도된 동작)

전체 문제 목록은 [LOCALIZATION/QA_CHECKLIST.md](LOCALIZATION/QA_CHECKLIST.md)를 참조하세요.

## 문제 해결

### 모드가 로드되지 않음
1. 게임을 완전히 재시작했는지 확인
2. Mods 폴더 경로가 올바른지 확인
3. 다른 번역 모드와 충돌하지 않는지 확인

### 일부 텍스트가 영어로 표시됨
- 절차적으로 생성되는 일부 콘텐츠는 아직 번역되지 않았을 수 있습니다
- [GitHub Issues](https://github.com/boramkim-cmd/korean_qud/issues)에 보고해 주세요

### 게임이 크래시됨
1. 최신 버전의 모드를 사용하고 있는지 확인
2. 다른 모드를 비활성화하고 테스트
3. 문제가 지속되면 [GitHub Issues](https://github.com/boramkim-cmd/korean_qud/issues)에 보고
