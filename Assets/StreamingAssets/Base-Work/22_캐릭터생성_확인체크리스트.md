# 🔍 캐릭터 생성부터 확인해야 할 체크리스트

새 게임을 시작할 때, 다음 순서대로 확인해주시면 됩니다.

---

## 1. 캐릭터 생성 화면 (Character Creation)
*   일반적인 메뉴나 설명이 한글로 나오는지 확인 (기존 번역 상태 점검)
*   **중요:** 캐릭터 닉네임을 지을 때 한글/영어 모두 입력해보기

## 2. 시작 장소 선택 (Start Location)
*   **⚠️ 필수:** `Joppa` 대신 **`Random Village`** (소금 사구, 언덕 등) 선택
*   이유: 랜덤 생성 콘텐츠(마을 이름, 역사, 퀘스트)를 즉시 테스트하기 위함입니다.

## 3. 게임 시작 직후 (Intro Log)
*   게임이 시작되자마자 **화면 오른쪽 메시지 로그**를 확인하세요.
*   **확인 포인트:**
    *   `=year=년 =month=월 =day=일, 당신은 ... 마을에 도착했습니다.` (한글 출력 여부)
    *   `지평선 너머로, 쿼드(Qud)의 정글이...` (한글 출력 여부)
    *   기존에는 이 부분이 영어(On the anniversary...)로 나왔었습니다.

## 4. 마을 진입 & NPC 이름
*   **NPC 이름:** 이제 **영어**로 나와야 정상입니다. (예: `Fayumet`, `Otho`)
    *   (NameGenFail 에러를 막기 위해 이름 생성 규칙을 원본으로 돌렸기 때문입니다)
*   **마을 이름:** 마을 이름도 **영어**로 나올 수 있습니다. (예: `Salt Dunes of ...`)

## 5. 촌장(Elder) 대화 & 퀘스트
*   촌장(노란색 NPC)에게 말을 거세요 (`Space` 키).
*   **조사 확인:** "촌장**이**", "당신**을**" 처럼 조사가 자연스러운지 확인.
    *   `촌장(이)가` 처럼 괄호가 보이면 안 됩니다.
*   **퀘스트 수락:** 퀘스트를 받고 `J` 키를 눌러 저널을 엽니다.
*   **퀘스트 제목:** 한글로 나오는지 확인 (예: `워터바인에 무슨 일이?` 또는 랜덤 퀘스트 제목)

## 6. 조각상/성소 (History)
*   마을에 있는 **조각상(Statue)**이나 **성소(Shrine)**를 찾아 `L` (Look) 키로 봅니다.
*   **역사 텍스트:** 
    *   `위대한 전투의 기념일에...` 같은 문구가 한글로 나오는지 확인.
    *   조사(`(이)가`, `(을)를`)가 자동으로 잘 변환되어 나오는지 확인.

---

### 요약: 정상 작동 기준
1.  **시작 로그:** 한글 ("지평선 너머로...")
2.  **NPC 이름:** 영어 (NameGenFail 없음)
3.  **퀘스트/역사:** 한글 + 자연스러운 조사
