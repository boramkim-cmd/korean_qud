# 퀘스트가 안 나오는 이유

**문제:** 테스트 퀘스트는 게임 시스템과 연결되지 않음

---

## 🎮 Caves of Qud 퀘스트 시스템

### 퀘스트가 나타나는 방법
1. **NPC와 대화** → 퀘스트 받음
2. **특정 장소 방문** → 퀘스트 활성화
3. **아이템 획득** → 퀘스트 트리거

### 우리가 만든 테스트 퀘스트
- ❌ 게임 시스템과 연결 안 됨
- ❌ 자동으로 나타나지 않음
- ❌ NPC나 이벤트 없음

---

## ✅ 해결: 실제 게임 퀘스트 번역

### 업데이트된 Quests.xml
```xml
1. "워터바인에 무슨 일이?"
   - 게임 시작 시 자동으로 받는 첫 퀘스트
   - 조파 마을에서 시작
   
2. "아르기브에게 고대고철 가져다주기"
   - 아르기브와 대화 시 받음
```

---

## 🧪 테스트 방법

### 1. 새 게임 시작
```
Main Menu → New Game
캐릭터 생성
조파(Joppa) 마을에서 시작
```

### 2. 퀘스트 확인
```
J 키 → 퀘스트 로그
"워터바인에 무슨 일이?" 확인
한글로 나오는지 확인
```

### 3. 예상 결과
```
✅ 성공:
퀘스트 제목: "워터바인에 무슨 일이?"
퀘스트 내용: "조파에서 북쪽으로..."

❌ 실패:
영어 그대로 표시
```
