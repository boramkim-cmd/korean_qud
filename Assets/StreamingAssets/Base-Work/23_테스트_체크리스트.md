## 🚀 게임 실행

### 2. Caves of Qud 실행
1. Steam 또는 직접 실행
2. Main Menu 도달 확인

### 3. Mods 메뉴 확인
```
Main Menu → Mods
```

**체크 사항:**
- [ ] "Korean Localization" 목록에 표시됨
- [ ] 체크박스 상태 확인
- [ ] 체크 안 되어 있으면 체크
- [ ] "Apply" 또는 "OK" 클릭
- [ ] 게임 재시작

## 🧪 테스트 1: 퀘스트 (필수)

### 4. 새 게임 시작
```
Main Menu → New Game → 아무 캐릭터 선택 → Joppa 시작
```

### 5. 퀘스트 로그 확인
```
게임 내에서 J 키 누르기
```

**확인 사항:**
- [ ] "워터바인에 무슨 일이?" 표시 (한글)
- [ ] "조파에서 북쪽으로..." 표시 (한글)
- [ ] 또는 "What's Eating the Watervine?" (영어) → 모드 미작동

**스크린샷 캡처:** 퀘스트 로그 화면

## 📊 테스트 4: 로그 분석

### 10. 게임 로그 확인
```bash
# 터미널에서 실행
tail -100 ~/Library/Application\ Support/com.FreeholdGames.CavesOfQud/Player.log | grep -i korean
```

**찾아야 할 메시지:**
```
[Korean Josa] v10 - Hotfix
[Korean Josa] MessageQueue Patch Active
```

**메시지 없으면:**
- JosaHandler.cs가 로드되지 않음
- manifest.json 확인 필요

## 🔧 문제 발견 시

### 시나리오 A: 모드가 Mods 메뉴에 없음
→ 모드 설치 문제
```bash
# 모드 재설치
cp -r "/Users/ben/Desktop/무제 폴더/StreamingAssets/Base-Work/Mod/KoreanLocalization" \
   ~/Library/Application\ Support/com.FreeholdGames.CavesOfQud/Mods/
```

### 시나리오 B: 모드는 있지만 번역 안 됨
→ 파일 문제 또는 게임 API 문제
- 로그 파일 확인
- manifest.json 확인
