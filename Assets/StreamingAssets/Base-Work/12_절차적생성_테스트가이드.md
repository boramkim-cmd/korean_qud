## 📋 테스트 파일

### 1. HistorySpice.json
```json
{
  "commonPhrases": {
    "leadIns": [
      "=year=년 =month=월, =name=(이)가 크롬 아치 아래를 걸었다"
    ]
  }
}
```

**테스트 시나리오:**
- 게임이 변수 치환: `"1025년 3월, Syuun(이)가..."`
- MessageQueue 패치: `"1025년 3월, Syuun이..."`

### 2. Naming.xml
```xml
<namestyle Name="Korean">
  <prefixes>
    <prefix Name="김" />
    <prefix Name="이" />
  </prefixes>
</namestyle>
```

**생성 예시:**
- `김민준`, `이서연`, `박지우`

## 🧪 테스트 방법

### 1. 파일 설치
```bash
cp Mod/KoreanLocalization/HistorySpice.json ~/Library/.../Mods/KoreanLocalization/
cp Mod/KoreanLocalization/Naming.xml ~/Library/.../Mods/KoreanLocalization/
```

### 2. 게임 내 확인

#### A. 역사 텍스트 (HistorySpice)
1. 새 게임 시작
2. 캐릭터 생성 완료
3. 게임 시작 후 메시지 확인
4. 역사 이벤트 발생 시 텍스트 확인

#### B. 절차적 이름 (Naming)
1. 새 NPC 생성 시 이름 확인
2. 절차적 아이템 생성 시 이름 확인

### 3. 로그 확인
```bash
tail -f ~/Library/Logs/Freehold\ Games/CavesOfQud/Player.log | grep "Korean Josa"
```

**예상 로그:**
```
[Korean Josa] Message: Syuun(이)가 → Syuun이
[Korean Josa] Message: 검(을)를 → 검을
```
