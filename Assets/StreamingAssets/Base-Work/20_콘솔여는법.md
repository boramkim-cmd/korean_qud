# 개발자 콘솔(Wish) 여는 법

**"₩" 키를 눌렀을 때 "restart turn"이 뜬다면 잘못된 키를 누르신 것입니다.**

Caves of Qud의 기본 콘솔 키는 **백틱(`)** 키입니다.

## 1. 올바른 키 위치
키보드 **숫자 1번의 왼쪽**에 있는 키를 눌러보세요.
*   영문 모드: **`** (백틱)
*   한글 모드: **₩** 또는 **~** (물결)

> **중요:** 한글 입력기 상태에서는 키가 인식되지 않을 수 있습니다. **한/영 키를 눌러 영문 상태로 바꾼 후** 눌러보세요.

## 2. 키 설정에서 변경하기 (확실한 방법)

만약 위의 키로도 열리지 않는다면, 직접 키를 설정해야 합니다.

1. **ESC**를 눌러 메뉴를 엽니다.
2. **Options** (옵션) 선택
3. **Key Mapping** (키 매핑) 선택
4. 아래로 스크롤하여 **"Debug"** 섹션을 찾거나 검색창에 **"Wish"**를 검색합니다.
5. **Permadeath/Wish** 항목을 찾습니다.
6. 해당 항목을 클릭하고 원하는 키(예: `F1` 또는 `F12`)로 변경합니다.
7. 저장하고 나옵니다.

## 3. Wish 명령어 사용법

콘솔이 열리면 화면 하단에 `>_` 입력창이 나타납니다. 여기에 명령어를 입력하세요.

*   `wish:고대의 검` (아이템 생성)
*   `wish:teleport 조파` (이동)

---

**요약:**
1. **한/영 키**를 눌러 영어 모드로 전환
2. **숫자 1 왼쪽 키(`)** 누르기
3. 안 되면 **옵션 > 키 매핑**에서 **"Wish"** 키 설정
