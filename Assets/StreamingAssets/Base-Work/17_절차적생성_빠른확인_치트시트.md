## 🎮 개발자 모드 활성화

### 콘솔 열기
```
` (백틱 키) 또는 ~ (물결표)
```

### Wish 명령어 형식
```
wish:명령어
```

---

## 🎲 1. 절차적 아이템 즉시 생성

### 기본 명령어
```
wish:고대의 검
wish:빛나는 방패
wish:저주받은 갑옷
wish:전설적인 반지
```

### 확인 사항
```
획득 메시지:
"고대의 검(을)를 획득했다!" → "고대의 검을 획득했다!" ✅
"빛나는 방패(을)를 발견했다!" → "빛나는 방패를 발견했다!" ✅
```

### 인벤토리 확인
```
1. i 키 (인벤토리)
2. 아이템 이름 확인
3. 아이템 설명 읽기
```

---

## 🗺️ 2. 절차적 지역 즉시 이동

### 텔레포트 명령어
```
wish:teleport 붉은 바위
wish:teleport 바나나 숲
wish:teleport 깊은 정글
```

### 확인 사항
```
이동 메시지:
"붉은 바위(으)로 이동했다" → "붉은 바위로 이동했다" ✅

지역 진입:
"바나나 숲 아케쉬(에) 도착했다" → "바나나 숲 아케쉬에 도착했다" ✅
```

### 월드맵 확인
```
Tab 키 → 빠른 여행 메뉴
→ 발견한 지역 목록에서 이름 확인
```

## 💬 3. 절차적 대사 즉시 확인

### NPC 소환
```
wish:spawn 장로 이루다드
wish:spawn 메멧
wish:spawn 워든 이라메
```

### 대화 시작
```
1. NPC 옆으로 이동
2. Space 또는 Enter
3. 대화 텍스트 확인
```

### 확인 사항
```
"조파(으)로 온 것(을)를 환영하네"
→ "조파로 온 것을 환영하네" ✅

"워터바인(을)를 키우지"
→ "워터바인을 키우지" ✅
```

---

## 📚 4. 책 내용 즉시 확인

### 책 생성
```
wish:book 명언집
wish:book 역사서
```

### 책 읽기
```
1. 인벤토리 (i)
2. 책 선택
3. r (read)
4. 내용 확인
```

### 확인 사항
```
책 내용:
"=name=(이)가 =item=(을)를 발견했을 때"
→ "파유멧이 프리즘을 발견했을 때" ✅
```

---

## 📜 5. 퀘스트 즉시 확인

### 퀘스트 시작
```
wish:startquest What's Eating the Watervine?
```

### 퀘스트 로그 확인
```
j 키 → 저널 → 퀘스트 탭
```

### 확인 사항
```
퀘스트 목표:
"붉은 바위(으)로 가라" → "붉은 바위로 가라" ✅
"시체(을)를 가져와라" → "시체를 가져와라" ✅
```

### 퀘스트 완료
```
wish:completequest What's Eating the Watervine?
```
