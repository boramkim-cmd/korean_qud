# 조파 마을 빠른 테스트 가이드

**작성일시:** 2026-01-13 11:33  
**목적:** 마을 내에서 조사 처리 즉시 확인

---

## ⚡ 1분 빠른 테스트

### 1단계: 게임 시작 (10초)
```
1. Caves of Qud 실행
2. "New Game" 선택
3. 기본 캐릭터 선택 (아무거나)
4. "조파" 시작 위치 확인
5. 게임 시작
```

### 2단계: 즉시 확인 (30초)
```
1. 게임 시작하자마자 화면 확인
   → 역사 텍스트에 조사 처리 확인

2. 북쪽으로 이동 (화살표 키)
   → 장로 이루다드 오두막

3. 장로와 대화 (Space 또는 Enter)
   → "조파(으)로 온 것(을)를 환영하네"
   → "조파로 온 것을 환영하네" ✅
```

### 3단계: 추가 확인 (20초)
```
1. 대화 선택지 확인
   → "이곳(이)가 조파인가요?"
   → "이곳이 조파인가요?" ✅

2. 응답 텍스트 확인
   → "워터바인(이)가 자랄 수 있고"
   → "워터바인이 자랄 수 있고" ✅
```

## 🎯 초간단 테스트 (30초)

### 방법 1: 장로 이루다드만
```
게임 시작 → 북쪽 이동 → 장로 대화 → 확인 완료!
```

**확인할 문장:**
- "조파(으)로 온 것(을)를 환영하네" → "조파로 온 것을 환영하네"
- "워터바인(을)를 키우지" → "워터바인을 키우지"
- "쿼드(에) 대해" → "쿼드에 대해"

### 방법 2: 메멧 퀘스트
```
게임 시작 → 남쪽 이동 → 메멧 대화 → "일거리(을)를 찾고 있습니다"
```

**확인할 문장:**
- "워터바인(을)를 먹고 있어" → "워터바인을 먹고 있어"
- "붉은 바위(으)로 가줄 수 있나?" → "붉은 바위로 가줄 수 있나?"
- "시체(을)를 가져와줘" → "시체를 가져와줘"


```
              ↑
              |
    [니마]  [입구]  [탐]
              |
              ↓
```

### 빠른 이동
