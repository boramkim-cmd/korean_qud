# Caves of Qud 한글화 테스트 상세 가이드

**목적:** 모든 번역 기능을 체계적으로 테스트

---

## 🎮 테스트 1: 기본 퀘스트 테스트 (가장 중요!)

### 캐릭터 생성
```
1. Caves of Qud 실행
2. Main Menu → Mods 확인
   - "Korean Localization" 체크되어 있는지 확인
   - 없으면 체크 후 게임 재시작
3. Main Menu → New Game
```

### 캐릭터 선택
```
아무거나 선택해도 됨 (추천: 빠른 테스트용)

추천 1: Arconaut (기본 캐릭터)
- 클릭 한 번으로 바로 시작 가능
- 조파(Joppa) 마을에서 시작

추천 2: Horticulturist
- 역시 빠른 시작 가능
```

### 시작 위치
```
Starting Location: Joppa (조파)
- 기본값 그대로 사용
```

### 게임 모드
```
Game Mode: Classic (기본값)
- Roleplay 모드는 테스트에 불필요
```

### 테스트 항목
```
1. 게임 시작 후 J 키 누르기 (퀘스트 로그)
2. 확인 사항:
   ✅ "워터바인에 무슨 일이?" ← 한글
   ✅ "조파에서 북쪽으로 2 파라상..." ← 한글
   ❌ "What's Eating the Watervine?" ← 영어 (실패)
```

**스크린샷:** 퀘스트 로그 화면 캡처

---

## 💬 테스트 2: NPC 대화 테스트

### NPC 찾기
```
조파 마을에서 다음 NPC와 대화:

1. Elder Irudad (장로 이루다드)
   - 위치: 마을 북쪽 건물
   - 이동: 화살표 키 또는 WASD
   - 대화: Space 키 또는 클릭

2. Mehmet (메멧)
   - 위치: 마을 남쪽 농장 (워터바인 근처)
   
3. Warden Yrame (워든 이라메)
   - 위치: 마을 중앙
```

### 테스트 항목
```
Elder Irudad와 대화 시작:

확인 1: 대화 텍스트
✅ "-음. 음? 여행자여?" ← 한글 (변수 처리됨)
⚠️ "-음. 음? =player.FormalAddressTerm=(이)여?" ← 변수 그대로 (문제)
❌ "Hmm. Hmm? Traveler?" ← 영어 (실패)

확인 2: 대화 선택지
✅ "이곳이 조파인가요?" ← 한글
✅ "여기서 물건을 파나요?" ← 한글
❌ "What is this place?" ← 영어 (실패)

확인 3: 조사 처리
✅ "조파로" ← 자연스러움
⚠️ "조파(으)로" ← 괄호 표시 (JosaHandler 미작동)
```

**스크린샷:** 
- 대화 시작 화면
- 대화 선택지 화면

## 🎲 테스트 3: 절차적 생성 테스트 (Random Village)

### 새 게임 시작
```
1. Main Menu → New Game
2. 캐릭터: 아무거나 (Arconaut 추천)
3. Starting Location: Random Village ⚠️ 중요!
   - Joppa 대신 Random Village 선택
4. Game Mode: Classic
5. Begin!
```

### 테스트 항목
```
게임 시작 직후 메시지 로그 확인:

확인 1: 마을 도착 메시지
✅ "키수 우에 2일, 당신은 모그푸르 마을에 도착했다" ← 한글
⚠️ "키수 우에 2일, you arrive at the village of Mogpur" ← 혼재
❌ "On the 2nd of Kisu Ue, you arrive at the village of Mogpur" ← 영어

확인 2: 역사 텍스트
✅ "지평선 너머, 쿼드의 정글이..." ← 한글
❌ "On the horizon, Qud's jungles strangle chrome steeples..." ← 영어
```

**스크린샷:** 
- 마을 도착 메시지
- 역사 텍스트 (Message Log 확인)

## 🏛️ 테스트 4: 역사 텍스트 테스트 (심화)

### 동상 조사
```
1. Random Village에서 시작
2. 마을 안 돌아다니기
3. 동상(statue) 찾기
4. 동상 조사 (Look 명령)
   - L 키 → 동상 선택
```

### 테스트 항목
```
동상 설명 텍스트:

✅ "=year=년, =name=이/가..." ← 한글 (변수 포함)
❌ "In =year=, =name= walked..." ← 영어
```

## 🎯 예상 결과 (현재 상태)

### 작동할 것 ✅
1. **퀘스트 텍스트** (Joppa)
   - "워터바인에 무슨 일이?"
   - "조파에서 북쪽으로..."

2. **기본 대화** (Joppa - 일부)
   - Elder Irudad, Mehmet, Warden Yrame
   - 선택지 한글 표시

### 문제 있을 것 ⚠️
1. **변수 처리**
   - `=player.FormalAddressTerm=(이)여?` ← 그대로 표시될 수 있음

2. **조사 처리**
   - "조파(으)로" ← 괄호 형태로 표시될 수 있음
   - JosaHandler.cs 미작동 가능성

3. **절차적 생성** (Random Village)
   - 일부 영어로 표시될 가능성
   - HistorySpice.json 부분 번역

### 작동 안 할 것 ❌
1. **대부분의 NPC 대화**
   - 조파 외 지역 NPC
   - 번역되지 않은 대화 ID

2. **UI 텍스트**
   - "New Game", "Continue" 등
   - 메뉴 텍스트
