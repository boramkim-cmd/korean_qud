## 📚 1. 책 내용 (Books.xml)

### 원본 예시
```xml
<book ID="Quotes" Title="Quotes">
  <page>
"Live and drink, friend. Now give me your water."
[[-Courteous bandit]]
  </page>
</book>
```

### 한글 번역 (조사 태그 포함)
```xml
<book ID="Quotes" Title="명언집">
  <page>
"살아라, 마셔라, 친구(여). 이제 네 물(을)를 내게 달라."
[[-예의 바른 산적]]
  </page>
  <page>
"=name=(이)가 =item=(을)를 발견했을 때, 세상(은)는 변했다."
[[-역사가의 기록]]
  </page>
</book>
```

**게임 실행 시:**
```
"파유멧(이)가 프리즘(을)를 발견했을 때, 세상(은)는 변했다."
↓ MessageQueue 패치
"파유멧이 프리즘을 발견했을 때, 세상은 변했다."
```

## 🗺️ 2. 지역명 (Naming.xml)

### 원본 구조
```xml
<namestyle Name="Qudish Site">
  <prefixes>
    <prefix Name="a" />
    <prefix Name="bi" />
  </prefixes>
  <postfixes>
    <postfix Name="kesh" />
    <postfix Name="ppa" />
  </postfixes>
  <templates>
    <template Name="Banana Grove *Name*" />
  </templates>
</namestyle>
```

### 한글 버전
```xml
<namestyle Name="QudishSiteKorean">
  <prefixes>
    <prefix Name="아" />
    <prefix Name="비" />
  </prefixes>
  <postfixes>
    <postfix Name="케쉬" />
    <postfix Name="파" />
  </postfixes>
  <templates>
    <template Name="바나나 숲 *Name*" />
  </templates>
</namestyle>
```

**생성 예시:**
- `바나나 숲 아케쉬`
- `바나나 숲 비파`

**사용 시:**
```
"바나나 숲 아케쉬(으)로 가라" → "바나나 숲 아케쉬로 가라"
"바나나 숲 비파(에)서 발견했다" → "바나나 숲 비파에서 발견했다"
```

## 📜 3. 역사 텍스트 (HistorySpice.json)

```json
{
  "commonPhrases": {
    "leadIns": [
      "In the month of =month= of =year=, =name= walked below the chrome arches"
    ]
  },
  "elements": {
    "glass": {
      "professions": ["glassblower", "window maker"],
      "materials": ["glass", "sand"],
      "mythicalEvent": ["a *var* shattered in every home"]
    }
  }
}
```

```json
    ],
    "epic": [
      "전설적인",

      "장대한",
      "위대한"
    ]

      "practices": ["거울 응시하기", "유리 불기", "프리즘(을)를 땅에 묻기"],
      "murdermethods": ["프리즘(으)로 가두어서", "유리(으)로 만든 단검(으)로"],

        "모든 집의 유리(이)가 산산조각났다",
```

```
입력: "전설적인 유리공예가(이)가 전설적인 프리즘(을)를 완성했다"
출력: "전설적인 유리공예가가 전설적인 프리즘을 완성했다"
```

## ⚔️ 4. 퀘스트 (Quests.xml)

### 절차적 퀘스트 예시
```xml
<quest Name="Find the Lost Artifact" 
       DisplayName="잃어버린 유물 찾기"
       Accomplishment="You found the =artifact= and returned it to =questgiver=."
       Hagiograph="In =month= of =year=, =name= discovered the legendary =artifact= in the depths of =location=.">
  
  <step Name="Travel to Location">
    <text>=location=(으)로 가라.</text>
  </step>
  
  <step Name="Find Artifact">
    <text>=artifact=(을)를 찾아라.</text>
  </step>
  
  <step Name="Return">
    <text>=artifact=(을)를 가지고 =questgiver=(에)게 돌아가라.</text>
  </step>
</quest>
```

**게임 실행 예시:**
```
변수 치환:
- location = "붉은 바위"
- artifact = "고대 프리즘"
- questgiver = "메멧"

결과:
"붉은 바위(으)로 가라" → "붉은 바위로 가라"
"고대 프리즘(을)를 찾아라" → "고대 프리즘을 찾아라"
"고대 프리즘(을)를 가지고 메멧(에)게 돌아가라" → "고대 프리즘을 가지고 메멧에게 돌아가라"
```

## 🎲 5. 아이템 이름 (절차적 생성)

### 아이템 이름 패턴
```xml
<!-- Naming.xml -->
<namestyle Name="KoreanArtifact">
  <prefixes>
    <prefix Name="고대의" />
    <prefix Name="전설적인" />
    <prefix Name="빛나는" />
    <prefix Name="저주받은" />
    <prefix Name="축복받은" />
  </prefixes>
  
  <suffixes>
    <suffix Name="검" />
    <suffix Name="방패" />
    <suffix Name="갑옷" />
    <suffix Name="반지" />
    <suffix Name="목걸이" />
  </suffixes>
</namestyle>
```

**생성 예시:**
- `고대의 검`
- `빛나는 방패`
- `저주받은 갑옷`

**사용 시:**
```
"고대의 검(이)가 빛났다" → "고대의 검이 빛났다"
"빛나는 방패(을)를 발견했다" → "빛나는 방패를 발견했다"
"저주받은 갑옷(은)는 위험하다" → "저주받은 갑옷은 위험하다"
```

## 👥 6. NPC 이름 + 대화

### NPC 이름 생성
```xml
<namestyle Name="QudishKorean">
  <prefixes Amount="1">
    <prefix Name="파" />
    <prefix Name="쿠" />
    <prefix Name="메" />
  </prefixes>
  
  <infixes Amount="0-2">
    <prefix Name="유" />
    <prefix Name="라" />
  </infixes>
  
  <postfixes Amount="1">
    <postfix Name="멧" />
    <postfix Name="슘" />
  </postfixes>
</namestyle>
```

**생성 예시:**
- `파유멧`
- `쿠라슘`
- `메유운`

### 대화 예시
```xml
<!-- Conversations.xml -->
<node ID="Greeting">
  <text>안녕, =player.name=(이)여. 나(은)는 =npc.name=(이)라고 한다.</text>
  <text>=quest.item=(을)를 찾고 있는가?</text>
</node>
```

**게임 실행:**
```
입력: "안녕, 시윤(이)여. 나(은)는 파유멧(이)라고 한다. 프리즘(을)를 찾고 있는가?"
출력: "안녕, 시윤이여. 나는 파유멧이라고 한다. 프리즘을 찾고 있는가?"
```

## 🌍 7. 종합 시나리오

### 게임 플레이 예시

```
[역사 생성]
"1025년 3월, 파유멧(이)가 붉은 바위(으)로 여행했다.
그곳에서 전설적인 프리즘(을)를 발견했다."

↓ MessageQueue 패치

"1025년 3월, 파유멧이 붉은 바위로 여행했다.
그곳에서 전설적인 프리즘을 발견했다."

[퀘스트 수락]
"파유멧(이)가 말했다: '프리즘(을)를 찾아주게.
바나나 숲 아케쉬(으)로 가면 찾을 수 있을 것이네.'"

↓ MessageQueue 패치

"파유멧이 말했다: '프리즘을 찾아주게.
바나나 숲 아케쉬로 가면 찾을 수 있을 것이네.'"

[아이템 발견]
"고대의 검(을)를 발견했다!"

↓ MessageQueue 패치

"고대의 검을 발견했다!"
```
