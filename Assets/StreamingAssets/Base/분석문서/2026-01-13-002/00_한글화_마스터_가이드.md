## 프로젝트 개요

### 게임 정보
- **게임명:** Caves of Qud
- **장르:** 로그라이크 RPG
- **특징:** 절차적 생성 시스템 (Procedural Generation)

### 한글화 범위
- 정적 콘텐츠: 대화, 퀘스트, 아이템, 스킬 등
- 동적 콘텐츠: 절차적으로 생성되는 역사, 이름, 이벤트

### 주요 도전 과제
1. **한글 조사 처리** (이/가, 을/를, 은/는)
2. **절차적 생성 시스템** 번역
3. **대량의 텍스트** (약 4-5 MB)
4. **복잡한 XML 구조** 유지

## 핵심 도전 과제

### 1. 한글 조사 처리

**문제:**
```
영어: "You found <item.name>"
한글: "당신은 <item.name>을/를 발견했다"
```

**해결 방안:**
- 게임 엔진 조사 처리 기능 확인
- 조사 회피 문장 구조 사용
- Python 스크립트 후처리

### 2. 절차적 생성 시스템

**파일:**
- HistorySpice.json (역사)
- Naming.xml (이름)
- LibraryCorpus.json (책)

**전략:**
- 변수 문법 유지
- 템플릿 번역
- 동적 조합 고려

### 3. XML 구조 유지

**주의사항:**
- 태그 변경 금지
- 속성 유지
- 중첩 구조 보존

### 4. 용어 일관성

**방법:**
- 용어집 작성
- 번역 메모리 활용
- 정기적 리뷰
