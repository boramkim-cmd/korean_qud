# 03-09. 나머지 절차적 생성 관련 파일 전략

---

## 03. LibraryCorpus.json

**절차적 생성 관여도:** ⭐⭐⭐⭐  
**크기:** 2.4 MB | **기간:** 50-100일 (선택적) | **역할:** 책 내용 생성

### 절차적 생성 방식
- 과학, 철학 텍스트 조각을 랜덤 조합
- 매 게임마다 다른 책 내용

### 번역 전략
- **선택적 번역 권장**
- 주요 책만 번역 (10-20개)
- 나머지는 영어 유지

---

## 04. Conversations.xml (동적 부분만)

**절차적 생성 관여도:** ⭐⭐⭐  
**크기:** 647 KB | **기간:** 5-7일 (동적 부분만) | **역할:** 동적 대화

### 절차적 생성 부분
```xml
<text><spice.gossip.leadIns.!random> <spice.gossip.twoFaction.!random></text>
```

### 번역 전략
1. HistorySpice.json 먼저 번역
2. 동적 대화 템플릿 찾기
3. 변수 유지하며 번역
4. 게임에서 테스트

---

## 05. Quests.xml (동적 부분만)

**절차적 생성 관여도:** ⭐⭐⭐  
**크기:** 27 KB | **기간:** 1일 (동적 부분만) | **역할:** 동적 퀘스트

### 절차적 생성 부분
```xml
<quest DisplayName="Find the <spice.adjectives.!random> <spice.nouns.!random>">
```

### 번역 전략
1. HistorySpice.json 먼저 번역
2. 동적 퀘스트 템플릿 찾기
3. 한글 어순에 맞게 조정

**예시:**
```xml
<!-- 원문 -->
"Find the <adjective> <noun>"

<!-- 번역 -->
"<adjective> <noun> 찾기"
```
