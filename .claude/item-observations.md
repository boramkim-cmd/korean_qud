# 아이템 관찰 기록

> 스크린샷/아이템 데이터 제공 시 여기에 누적 기록
> 개발 검증 체크리스트로 활용

## 기록 형식

```
### YYYY-MM-DD — [출처: 스크린샷/위시/테스트 등]

| 원문 | 기대 번역 | 실제 결과 | 패턴 | 상태 |
|------|----------|----------|------|------|
| electrobow | 일렉트로보우 | ? | DirectMatch | 미확인 |

패턴: DirectMatch / PrefixSuffix / Partial / Fallback / Miss
상태: OK / 버그 / 미확인
```

---

(아직 기록 없음 — 다음 스크린샷/아이템 데이터 제공 시 추가)
