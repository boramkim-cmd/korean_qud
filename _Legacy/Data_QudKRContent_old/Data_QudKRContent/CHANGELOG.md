# Changelog

모든 주요 변경사항이 이 파일에 기록됩니다.

## [0.2.0] - 2026-01-15

### Added
- 🎉 **새 프로젝트 시작**: 모듈화된 구조로 완전히 재작성
- ✅ **핵심 시스템**:
  - `ScopeManager`: Stack 기반 범위 관리
  - `TranslationEngine`: 색상 태그, 체크박스, 대소문자 처리
  - `ModEntry`: 에러 핸들링 및 초기화
- 📁 **모듈 구조**:
  - `00_Core/`: 핵심 엔진
  - `01_Data/`: 번역 데이터 (화면별 분리)
  - `02_Patches/`: Harmony 패치 (기능별 분리)
  - `99_Utils/`: 유틸리티
- 📚 **문서화 시스템**:
  - `Docs/BugReports/`: 버그 보고서
  - `Docs/Solutions/`: 해결책 문서
  - `KNOWN_ISSUES.md`: 알려진 문제
  - `CHANGELOG.md`: 변경 사항

### Fixed
- 🐛 딕셔너리 중복 정의 문제 해결 (파일 분리)
- 🐛 전역 패치 범위 문제 해결 (Stack 기반 관리)
- 🐛 원본 메서드 검증 추가
- 🐛 데이터 중복 제거 (계층 구조)
- 🐛 색상 태그 처리 개선 (Qud 형식 지원)

### Changed
- 🔄 데이터와 로직 완전 분리
- 🔄 전역 패치 최소화, 화면별 패치 우선
- 🔄 점진적 개선 방식 도입 (Phase별 구현)

### Removed
- ❌ 중복된 딕셔너리 정의
- ❌ 불필요한 전역 패치

## [0.1.0] - 2026-01-14 (구 프로젝트)

### Issues

- 전역 패치로 인한 범위 제어 문제
- 원본 메서드 확인 부족
- 문서화 부족

> [!NOTE]
